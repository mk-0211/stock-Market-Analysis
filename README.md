This project is a high-fidelity financial analytics dashboard designed to monitor and evaluate equity performance using advanced risk-adjusted metrics. It provides a deep dive into price trends, volatility regimes, and correlation structures for a portfolio including AAPL, GOOG, MSFT, and NFLX.

🚀 Key Financial Insights
Performance Metrics: Achieved a Cumulative Return of +12.41% with an Annualized Volatility of +22.47%.
Risk-Adjusted Returns: The selected portfolio (focused on AAPL) shows a Risk-Adjusted Return of 2.26, indicating that higher volatility has been effectively rewarded with superior performance.
Volatility Analysis: Tracked Rolling Volatility (20D and 200D) to identify market regimes, currently categorized as a "High Volatility" regime.
Drawdown Management: Monitored Monthly Max Drawdown, reaching a peak negative excursion of -0.06%, essential for downside risk assessment.
Correlation Matrix: Analyzed inter-stock relationships, such as the 0.63 correlation between AAPL and GOOG, to support portfolio diversification strategies.

🛠️ Technical ImplementationTool:
Power BI.Key Measures (DAX):
Rolling Volatility: 20-day and 200-day standard deviation of returns.
Moving Averages: 20-day MA to identify price momentum.
Risk Metrics: Calculation of Max Drawdown and Risk-Adjusted Return ratios.
Visualizations: Area charts for volatility trends, dual-axis bar/line charts for return vs. risk, and heatmaps for ticker correlation.

📈 Executive Summary
The analysis confirms that while the portfolio experiences high volatility, the 2.26 Risk-Adjusted Return justifies the current strategy. Monitoring the Rolling Return (20D) by month shows a significant recovery trend starting from March through May 2023.
